# US State Data Visualization (Population & GDP) – Power BI Project

## Project Overview

This project presents an interactive data visualization of United States state-level data, focusing on:
- Population
- Gross Domestic Product (GDP)

The goal is to provide clear, insightful, and interactive dashboards to explore economic and demographic patterns across US states using Power BI.
## Objectives
- Visualize population distribution across US states
- Compare GDP across states 
- Identify relationships between population and economic output
- Enable interactive exploration for better decision-making
## Tools & Technologies Used
- Power BI – Data visualization and dashboard creation
- GitHub – Version control and project sharing
- Dataset Sources – US state population and GDP datasets

## Dashboard Features
### 1. Population Visualization
- Census Region wise population distribution using map
- Bar charts showing count of state in each census region
- line chart shows the population growth rate for top 5 state
- Table with basic information of state and population
![visuals/dashboard](https://github.com/birendra-bit/us-population-and-gdp-visualization/blob/main/visuals/dashboard-population-visualization.png)

### 2. GDP Visualization
- State-wise GDP growth trend over the years
- GDP per capita by census region wise
- sum of gdp state wise (top 5 state)
